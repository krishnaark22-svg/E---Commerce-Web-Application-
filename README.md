🛒 E-Commerce Web Application
A simple full-stack E-Commerce Web Application that allows users to browse products, add items to a cart, and place orders. This project demonstrates core concepts of frontend development, backend APIs, and database integration.
🚀 Features
🛍️ Product listing and catalog
🛒 Add to cart functionality
💳 Checkout system
🔐 User authentication (Login/Register)
👨‍💼 Role-based access (Admin/User)
📦 Order management
🔗 RESTful APIs
🗄️ Database integration (MongoDB / MySQL / PostgreSQL)
🧰 Tech Stack
Frontend:
HTML
CSS
JavaScript
Backend:
Node.js
Express.js
Database:
MongoDB (default)
Authentication:
JSON Web Token (JWT)
📡 API Endpoints
🔹 Products
GET /api/products → Get all products
POST /api/products → Add new product
🔹 Authentication
POST /api/auth/register → Register user
POST /api/auth/login → Login user
🔹 Orders
GET /api/orders → Get all orders
POST /api/orders → Place order
🎯 Learning Outcomes
Understand full-stack application flow
Learn REST API development
Work with databases and backend integration
Implement authentication and authorization
Build real-world project structure
